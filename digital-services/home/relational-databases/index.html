Relational Databases with SQL

In order to acquire data from customers, businesses need databases to store and manipulate data.

Relational databases are the most common type of database structure. Popular database systems such as Oracle, MySQL, and Microsoft SQL Server, etc. all use relational models to store data.

As data continue to expand and evolve, we need more sophisticated technology to process Big Data. My research includes database systems and the SQL language, which can be used as a Data Definition Language (DDL) and a Data Maniplulation Language (DML).
