Python for Data Science

17 November 2022

It is recommended to connect to a VPN when visiting this site from China as certain assets require connection to censored websites to load.

Overview

Welcome to the course website of Python for Data Science. In this course, we will learn object-oriented programming with Python. Python is a high-level, general-purpose programming language. It is popular because of its English-like syntax and applications in multiple use cases.

Our focus in this course will be Python for data science and automation. You will learn the basics of Python, including the syntax, data structure, and algorithms. There will be assignments, big and small, designed to help you grasp the concepts and apply Python in data analysis.

Prerequisites

This course assumes no prior knowledge of Python programming. It is designed for beginners who‚Äôre interested in using Python to analyse data and professionals who‚Äôre looking to refresh their scientific programming skills. The course assumes high school level statistics knowledge.

Materials

Python Crash Course (3rd Edition) will be the textbook used in this course. We will also use resources from other books, online forums and videos in this course as well. We will use credible learning resources coupled with methodical learning design to acquire Python for data science knowledge and skills.

Students should install the latest version of Python from the official Python Software Foundation website at https://python.org and Anaconda https://anaconda.org. We will use Jupyter Notebook EXTENSIVELY in this course.

Approach

This course is self-paced, meaning that students have the freedom to learn the material on their own. However, it is encouraged to form study groups to ensure progress. The course syllabus (attached below) details the structure of the course, including how students can make the most out of learning activities and how learning outcome should be measured.

Students will read the textbook, complete exercises and participate in additional learning activities to learn the material.
